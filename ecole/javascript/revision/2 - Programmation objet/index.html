Exercice de révision - Programmation objet :
  1. Programmez la classe 'Zombie' selon la méthode vue en classe.
  2. Créez deux instances de la classe Zombie. Chaque instance devra contrôler 1 zombie (ses déplacements).
  3. Puisque les zombies se déplaceront, ajoutez le code des animations que nous avons vu en classe (tick + window.requestAnimationFrame).
  4. Chaque zombie doit se déplacer sur l'axe des X aléatoirement. Il n'y a pas de collision entre les zombies (ils peuvent se chevaucher).
  5. Les zombies doivent être sur le sol, donc leurs pieds doivent être près de la bordure blanche inférieure.
  6. Les zombie ne doivent pas sortir du carré blanc.
  7. Lorsque l'on clique dans le carré blanc, une paire de yeux (eyes.gif) apparait à l'endroit cliqué. Celle-ci disparait lentement et est éventuellement retirée de la liste des sprites. Pour la faire disparaitre, utilisez le style 'opacity'.
Notes : Vous pouvez modifier le HTML. Ajoutez votre code dans un fichier JavaScript à part. Programmez et testez un numéro à la fois. Vous pouvez ajouter et utiliser jQuery si cela vous aide.
