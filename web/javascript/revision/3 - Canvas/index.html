Exercice de révision - Canvas :
  1. Pour cette révision, ne créez pas de DIV ni aucun autre élément HTML!
  2. Ajoutez la logique d'animation que nous avons vu en classe.
  3. Créez vous la classe NinjaGaden et ajoutez-la à la liste des sprites.
  4. Le sprite NinjaGaden doit contrôler un petit ninja (voir images/ninja.png).
  5. Le ninja doit être afficher sur le sol devant le monstre.
  6. L'usager doit pouvoir contrôler le ninja, donc le faire bouger de gauche à droite via les touches A et D.
  7. Créez la classe NinjaStar. Lorsque l'usager appuie sur la touche espace, une nouvelle instance de NinjaStar est créée et ajoutée à la liste des sprites. Celle-ci doit apparaitre (star.png) et se diriger vers le monstre.
  8. L'étoile est supprimée de la liste des sprites lorsqu'elle dépasse la limite du canvas.
Notes : Vous pouvez modifier le HTML. Ajoutez votre code dans un fichier JavaScript à part. Programmez et testez un numéro à la fois. Vous pouvez ajouter et utiliser jQuery si cela vous aide.
