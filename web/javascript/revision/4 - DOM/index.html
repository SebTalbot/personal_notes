Exercice de révision - DOM :
  1. Lorsque l'on clique sur "Ajouter", un ou plusieurs paragraphes sont ajoutés à la page, selon le nombre voulu. Les paragraphes sont ajoutés sous le rectangle blanc.
  2. Le texte de chaque paragraphe est 'ceci est un nouveau paragraphe'.
  3. Lorsque l'on clique sur l'un des paragraphes ajoutés, celui-ci est retiré du document.
Notes : Vous pouvez modifier le HTML. Ajoutez votre code dans un fichier JavaScript à part. Programmez et testez un numéro à la fois. Vous pouvez ajouter et utiliser jQuery si cela vous aide.
